#Git 강의

##1주차
###키워드 정리
Git - 프로젝트의 버전 관리를 위한 도구
GitHub - Git 프로젝트의 원격 저장소 + 커뮤니티 기능

sourcetree - git을 편하게 사용할 수 있게 해주는 프로그램


commit - 현재 프로젝트 상태 저장
push - 원격 저장소에 로컬 저장소의 내용을 밀어 넣는 것(업로드?)
pull - 원격 저장소에서 로컬 저장소로 내용을 댕겨 오는 것(다운로드?)

clone - 원격 저장소에 있는 내용들을 복제

###Git 꿀팁
pull - commit(로컬) - push


###총평
1주차 - Git과 Git Hub는 학부 때부터 계속 알고있던 개념이지만 다시 공부를 하면서 그 내용들이 구체화되었다.
